CompanyFleetShare – Face Verification API

This deployment is API-only. Your main app should call these endpoints:

  • POST /api/session-create (form-data: license_image) → returns session_id
  • POST /api/session-verify (form-data: session_id, selfie_image) → returns match result

If you see a 404 when opening /, it just means there is no UI. This page is here to confirm the deployment is alive.
