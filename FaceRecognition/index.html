Face recognition service

Not runnable on Vercel serverless. Dependencies (dlib, MediaPipe, OpenCV) exceed the ~500 MB Lambda limit. Deploy this folder with Docker instead (Render, Railway, Fly.io, VPS).

Use the included Dockerfile:

docker build -t face-api .
docker run -p 8000:8000 face-api

API runs on port 8000 (e.g. /match, /verify-license-face per app.py).

Point your Vercel session project’s FACE_VALIDATOR_URL at this public URL.
